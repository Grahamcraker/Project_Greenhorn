Detailed Description

Main class for handling screens. This class keeps a list of screens representing what is drawn on screen. Only one screen will be drawn unless a popup has appeared over top of another screen. This class also makes calls to have the screens update and draw. The screen manager will only allow the top screen to handle input from the player.
